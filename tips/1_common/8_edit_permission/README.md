# Edit 권한 요청

## 권한을 요청하는 User
{: .left-bar-title }
1. 권한이 필요한 Project나 3rd Party의 Project Information, 3rd Party Information 화면의 우측 상단의 "Request Permission" 버튼을 클릭합니다.
<img src="../../images/common/edit_permission/request.png" alt="select" class="styled-image">
2. 만약, 권한 요청을 하였지만 필요가 없어진 경우 Creator/Editor의 승인 전에 "Cancel Request Permission" 버튼 클릭을 통해 취소할 수 있습니다.
<img src="../../images/common/edit_permission/cancel.png" alt="select" class="styled-image">

## Project/3rd Party의 Creator/Editor
{: .left-bar-title }
1. User가 권한을 요청한 경우, 권한 요청 메일이 발송됩니다.
<img src="../../images/common/edit_permission/request_mail.png" alt="select" class="styled-image">
2. 메일의 Approve/Reject를 클릭하거나, 해당 project 나 3rd party의 information에 진입시 아래와 같은 팝업이 뜨게 됩니다.
<img src="../../images/common/edit_permission/request_project_information.png" alt="select" class="styled-image">
3. Approve를 클릭시 Edit 권한이 부여되고, Reject 클릭시 권한이 부여되지 않습니다.
